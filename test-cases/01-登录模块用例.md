## ✅ 测试用例

### TC-LOGIN-001：正确账号密码登录
- **前置条件**：已有注册账号 testuser / Test@123
- **测试步骤**：
  1. 打开登录页面
  2. 输入用户名：testuser
  3. 输入密码：Test@123
  4. 点击"登录"按钮
- **预期结果**：登录成功，跳转到首页，显示用户名

### TC-LOGIN-002：错误密码登录
- **前置条件**：已有注册账号 testuser
- **测试步骤**：
  1. 打开登录页面
  2. 输入用户名：testuser
  3. 输入错误密码：WrongPass
  4. 点击"登录"按钮
- **预期结果**：提示"用户名或密码错误"，停留在登录页

### TC-LOGIN-003：空用户名登录
- **前置条件**：无
- **测试步骤**：
  1. 打开登录页面
  2. 用户名留空
  3. 输入任意密码
  4. 点击"登录"按钮
- **预期结果**：提示"请输入用户名"

### TC-LOGIN-004：空密码登录
- **前置条件**：无
- **测试步骤**：
  1. 打开登录页面
  2. 输入任意用户名
  3. 密码留空
  4. 点击"登录"按钮
- **预期结果**：提示"请输入密码"

### TC-LOGIN-005：SQL注入测试
- **前置条件**：无
- **测试步骤**：
  1. 打开登录页面
  2. 输入用户名：`' OR '1'='1`
  3. 输入密码：`' OR '1'='1`
  4. 点击"登录"按钮
- **预期结果**：登录失败，不暴露数据库错误信息
