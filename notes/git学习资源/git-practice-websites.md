## ⭐ 推荐：Git 交互式教程

**[Git Interactive Tutorial](https://woyeyao.github.io/Git-Interactive-Tutorial/)**

一个可视化的 Git 交互学习网站，通过图形化界面帮助你直观理解 Git 的核心概念：

- 📊 **可视化**：每一步操作都能看到 commit 树的变化
- 🎯 **交互式**：边学边练，即时反馈
- 🧩 **覆盖全面**：从基础到进阶的 Git 操作
- 🌐 **在线使用**：无需安装，浏览器打开即用

### 适合练习的内容：

| 模块 | 内容 |
|------|------|
| Git Commit | 提交操作 |
| Git Branch | 分支管理 |
| Git Merge | 合并分支 |
| Git Rebase | 变基操作 |
| Git 移动 | 相对引用与 HEAD 移动 |
| Git 撤销 | reset / revert 等撤销操作 |
| 远程仓库 | push / pull / fetch 等 |
